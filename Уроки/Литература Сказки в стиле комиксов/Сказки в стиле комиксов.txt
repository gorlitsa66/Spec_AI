### **Урок по литературе для 5 класса: «Сказки в стиле комиксов»**  
**Цель:** Научить учащихся анализировать текст через визуализацию, создать комикс по мотивам сказки.  

**Оборудование:**  
- Текст сказки «Колобок» (распечатки)  
- Шаблоны комиксов (пустые раскадровки)  
- Цветные карандаши/фломастеры  
- Примеры комиксов по сказкам  
- Мультимедиа: презентация с этапами создания комикса  

---

### **1. Введение (5 минут)**  
**Приём «Две картинки»:**  
- Показываем иллюстрацию к сказке и кадр из комикса:  
  *«Чем отличаются эти изображения одной сказки?»*  

**Проблемный вопрос:**  
*«Можно ли рассказать сказку без слов, только картинками?»*  

**Актуализация знаний:**  
- Мини-дискуссия: *«Какие элементы важны в комиксе?»* (персонажи, последовательность, эмоции)  
- Показ 2-3 слайдов с примерами удачных комиксов по сказкам  

---

### **2. Основная часть (20 минут)**  

#### **1. Анализ текста (7 минут)**  
- Чтение сказки «Колобок» по ролям  
- Выделяем ключевые сцены (6-8 моментов):  
  1. Появление Колобка  
  2. Встреча с Зайцем  
  3. ...  
  4. Финал  

#### **2. Практическая работа (13 минут)**  
**Этапы создания комикса:**  
1. **Раскадровка:** Распределяем сцены по шаблонам (каждый ученик выбирает 1-2 сцены)  
2. **Визуализация:**  
   - Прорисовка персонажей с эмоциями (шаблоны лиц: радость, страх)  
   - Добавление «облаков» для реплик  
3. **Оформление:**  
   - Вариант: объединить работы в общий комикс на доске  

**Пример диалога для вставки:**  
- Заяц: *«Колобок, я тебя съем!»*  
- Колобок: *«Не ешь меня, спою тебе песенку!»*  

---

### **3. Заключение (5 минут)**  
**Галерея работ:**  
- Каждая группа показывает свой фрагмент, рассказывает сцену  
- Голосование за самый выразительный комикс  

**Рефлексия:**  
- *«Что было сложнее: нарисовать или выбрать главное в тексте?»*  
- *«Какие сказки ещё хотели бы превратить в комикс?»*  

**Домашнее задание (на выбор):**  
1. Дорисовать незаконченные сцены  
2. Создать комикс по сказке «Репка» (3-4 кадра)  

---

### **4. Ответы на вопросы (10 минут)**  
**Разбор трудностей:**  
- Как показать движение на рисунке? (штрихи, размытость)  
- Что делать, если не умеешь рисовать? (использовать символы, схематичные фигуры)  

**Игра «Угадай сцену»:**  
- Учитель показывает случайный кадр из работ — ученики называют эпизод  

**Совет:**  
- Для следующих уроков: использовать цифровые tools (Canva, Pixton)  

---

**Методические рекомендации:**  
- Для слабых учеников: подготовить шаблоны персонажей  
- Важно: акцент на передачу смысла, а не на художественное мастерство  
- Возможная адаптация: создать комикс-кроссворд по сюжету  

Урок развивает креативное мышление и глубокое понимание текста через визуальное восприятие!
